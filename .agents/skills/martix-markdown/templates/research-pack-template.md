# Research pack template

## Purpose

Use this template when preparing a research memo that feeds future grouped
markdownlint rules, overlay bridges, or toolchain comparison work.

## Recommended outline

1. Scope, desired outcome, and non-goals
1. Approved source families and precedence model
1. Existing package surfaces to reuse
1. Candidate grouped domains and target files
1. Upstream rule IDs, aliases, tags, and option surfaces
1. Repo-local overlay notes and conflicts
1. Toolchain or formatter interaction notes
1. Gaps, open questions, and next actions

## Review prompts

- Which approved upstream markdownlint sources are authoritative for the topic?
- Which repo-local overlays materially change the advice?
- Does the outcome belong in a grouped rule file, a reference memo, a template,
  or a config sample?
- Which planned domain in [taxonomy.json](../assets/taxonomy.json) does this
  research unlock or reshape?
